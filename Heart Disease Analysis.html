  • age: Age of the individual
  • sex: Sex of the individual (1 = male; 0 = female)
  • chest pain type: Type of chest pain experienced by the individual (4 values)
    • Value 0: Typical angina
    • Value 1: Atypical angina
    • Value 2: Non-anginal pain
    • Value 3: Asymptomatic
  • trestbps: Resting blood pressure in mm Hg on admission to the hospital
  • chol: Serum cholesterol in mg/dl
  • fbs: Fasting blood sugar > 120 mg/dl (1 = true; 0 = false)
  • restecg: Resting electrocardiographic results
    • Value 0: Normal
    • Value 1: ST-T wave abnormality (e.g., T wave inversions and/or ST elevation or depression of > 0.05 mV)
    • Value 2: Probable or definite left ventricular hypertrophy by Estes' criteria
  • thalach: Maximum heart rate achieved
  • exang: Exercise-induced angina (1 = yes; 0 = no)
  • oldpeak: ST depression induced by exercise relative to rest
  • slope: Slope of the peak exercise ST segment
    • Value 1: Upsloping
    • Value 2: Flat
    • Value 3: Downsloping
  • ca: Number of major vessels (0-3) colored by fluoroscopy
  • thal: Thalassemia (heart condition)
    • Value 3: Normal
    • Value 6: Fixed defect
    • Value 7: Reversible defect
  • target: Diagnosis of heart disease (output)
    • Value 0: Less chance of heart attack
    • Value 1: More chance of heart attack
